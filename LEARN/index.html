Setting Up Your Development Environment

Before you start coding, you need to set up your environment. To do this you will need to install several software packages to your computer. See videos of how to install these tools. Here you will install node (which automatically installs npm too), git (which automatically installs gitBash too). You will also need to install an IDE - we recommend Visual Studio Code.

Start with the Basics

Start with HTML 5 which teaches you the basic building blocks for displaying content on a page. Next learn CSS 3 so that you know how to edit the styles of fonts, etc. Afterwards learn the basics of Javascript.
